Configuring and Initialization

Step 5: Configure your username and Password

Intorduce yourself by configuring your username and email to the Git. you can do this by following commands:

  1. git config --global user.name "your_name_here"
  2. git config --global user.email "your_email@email.com"

Step 6: Initialize Local Github Repo

Initialize your Git Local repo by using the "git init". After Initialization a hidden .git folder will the created

  • git init
