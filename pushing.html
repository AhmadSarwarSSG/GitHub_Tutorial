Connecting to Remote and Pushing

Step 9: Connecting to Remote

Now, it's time to connect your computer to GitHub with the command provided by the GitHub Repo you can also say it as the URL of your repo.

  1. git remote add origin https://github.com/your_username/Demo.git

Step 10: Pushing

Now that we have added the remote, we can push our code to GitHub.com. Once you have done pushing your code will be at origin and you can view it in your repo.

  • git commit -m "Your Message"
