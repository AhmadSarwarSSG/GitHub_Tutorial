Adding and Committing

Step 7: Stagging Files

Then, to tell the Git program you care about this file and want to track any changes from this point forward, enter:

  1. git add "Your Filename Here""
  2. git add * (It'll stage all the files)

Step 8: Committing

Every time you accomplish some work, you can write a Git commit to store that version of your file, so you can go back later and see what it looked like at that point in time. Whenever you make a change to your file, you create a new version of that file, different from the previous one. To make a commit, enter:

  • git git push origin master"
