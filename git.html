Installing Git Locally

Step 3: Download and Install Git

Visit the Github Website and download the Git for your PC. If you have windows OS you have download Git for Windows otherwise you have to install download Git for MAC. After downloading the Git install it on your PC.

Step 4: Accessing the Git

Create a new folder where you want to manage your repository locally on your PC. Open the folder an right Click there you'll see the "Git GUI" and "Git Bash" options go for "Git Bash". Consle Screen will be opened.
